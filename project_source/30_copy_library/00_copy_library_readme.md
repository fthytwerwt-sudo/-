# 文案知识库说明

`project_source/30_copy_library/` 用于保存《视频工厂》后续可持续维护的文案知识库底稿与演进记录。

## 当前已收录内容

- `01_gold_samples_raw.md`：用户当前“喜欢”的 10 条原始文案样本
- `02_anti_patterns_raw.md`：用户当前“不喜欢”的 10 条原始文案样本
- `08_update_log.md`：文案知识库目录内的更新记录

## 当前目录状态

- 本轮只完成原始样本入库与最小骨架初始化
- 当前样本均保留原始顺序与原始内容
- 当前尚未做结构化整理、规则提炼、标签分类或优劣分析

## 后续可继续补充的内容

- 喜欢的片段
- 口语原话
- 反例片段
- 组件库或可复用表达块

一句话说明：本目录用于后续文案自动化、口语化校准、反例识别与生成质量提升，但本轮只完成原始入库，不做规则化整理。
