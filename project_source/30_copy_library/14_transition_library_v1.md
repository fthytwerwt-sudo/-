# 过渡句库 v1

## 当前主结论

- 当前用户偏好的过渡句，不像写文章，更像口播时顺嘴把人往下带
- 最稳的过渡，不是华丽辞藻，而是 `问题推进`、`举例切换`、`编号展开`
- 当前项目最值得先复用的，是能把逻辑往前推的短桥

## 使用边界

- 本文件只基于当前“喜欢样本”得出
- 当前属于初步判断，后续可继续补强
- 这里整理的是“桥的动作”，不是逐字固定台词

## 1. 问题推进型

- 典型表达：
  - `那问题来了`
  - `那怎么办呢`
  - `为什么`
  - `你要想想`
- 这类桥的作用：
  - 把下一段内容顺出来
  - 让听众自然进入下一层推理
- 当前项目适合复用：
  - 解释型脚本
  - 判断型脚本
  - 结构转折处

## 2. 举例切换型

- 典型表达：
  - `举个例子`
  - `最后说一个真实的……`
  - `比如说`
- 这类桥的作用：
  - 从抽象判断切到具体场景
  - 让内容不空
- 当前项目适合复用：
  - 工具说明
  - 框架解释
  - 方法演示

## 3. 编号展开型

- 典型表达：
  - `第一个……第二个……`
  - `第一个问题……第二个问题……`
- 这类桥的作用：
  - 让结构一眼清楚
  - 很适合做信息密度高但要口语化的内容
- 当前项目适合复用：
  - 技巧盘点
  - 选择标准
  - 流程拆解

## 4. 纠偏 / 让步型

- 典型表达：
  - `但`
  - `不过`
  - `其实`
  - `问题在于`
- 这类桥的作用：
  - 避免内容变成一条直线
  - 让判断更像真人思考，不像背稿
- 当前项目适合复用：
  - 对比内容
  - 风险提醒
  - 方案取舍

## 5. 悬念回扣型

- 典型动作：
  - 先扔一个问题
  - 中间解释
  - 再回头说“如果这件事聊明白，你就能理解前面那个问题”
- 样本参考：`喜欢 3`
- 这一类桥的特点：
  - 很强
  - 但更适合长版
- 当前判断：
  - 保留观察价值
  - 先不作为当前主版默认桥

## 当前最适合先复用的过渡桥

- 第一优先：`问题推进型`
- 第二优先：`举例切换型`
- 第三优先：`编号展开型`
- 补充位：`纠偏 / 让步型`
- 观察位：`悬念回扣型`

## 当前一句话理解

- 当前用户喜欢的过渡，不是漂亮，而是顺
- 只要一句桥能把“下一步为什么要讲”带出来，它就已经很有价值了
