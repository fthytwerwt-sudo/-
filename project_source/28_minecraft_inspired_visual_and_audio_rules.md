# Minecraft-inspired 视觉与声音规则

## 1. 文件定位
本文件只负责 4 件事：
- 定义 vNext 的统一视觉外壳
- 定义 `Minecraft-inspired` 的合法边界
- 规定开头人物壳 / 结尾总结壳 / `Prompt 引用尾卡` 的统一风格
- 给出声音路线建议与参数建议

## 2. inspired 合法边界
当前只允许：
- `Minecraft-inspired 原创体素方块风`
- 原创体素角色
- 原创体素工作台环境
- 原创体素 UI 面板
- 原创体素色板与几何块面

当前明确禁止：
- 直接复用官方 `logo`
- 直接复用官方 `fonts`
- 直接复用官方 `textures`
- 直接复用官方 `images`
- 直接复用官方 `models`
- 直接复用官方 `sounds`
- 做成让用户误解为官方授权、官方联名或官方素材包

## 3. 当前统一视觉外壳规则
- 开头人物壳默认统一为：`Minecraft-inspired 原创体素方块风`
- 结尾总结壳默认统一为：`Minecraft-inspired 原创体素方块风`
- `Prompt 引用尾卡` 默认共用同一套体素面板语言
- 三者都只承担“壳”和“引用”功能，不替代中段证据

## 4. 默认声音路线
- 目标感觉：平静、轻陪伴、低压、像游戏向导
- 默认避免：播报腔、销售腔、新闻腔、过度热情
- 参数建议：
  - 语速：`0.92 - 1.0`
  - 句间停顿：略长于默认
  - 情绪强度：低到中等
- 当前状态边界：
  - 只代表声音路线建议
  - 不代表已经选定最终 TTS 模型
  - 不代表已经完成内容验证
