# 主题
别再瞎用豆包了。先别让它直接写，先让它帮你搭框架、做整理、做优化，今天就能上手。

## 基础参数
- 总时长：72秒
- 视频比例：9:16

## 目标场景
AI 职场任务现场拆解 / AI 问题点破

## 目标用户
已经在用豆包，但总觉得“看着厉害、最后还得自己改”的职场用户；希望今天就能抄走动作，不想再听抽象理念。

## 全局质量要求
- 开头与结尾继续由 API 人物段承担判断与收束，但人物视觉统一改成 Q版 3D 卡通人形 IP
- 中段主体必须由真实豆包录屏承担，不能退化成纯 PPT / 纯卡片说明
- 至少清楚出现 3 个动作位：先搭框架、先做整理、只做优化 / 纪要变待办
- 轻量 `steps_card` 只负责压实口诀与最小动作，不抢主体
- 静音看也能读出“问题 → 动作 → 结果”的推进
- 卡通人物必须可信、克制、可爱但不低幼，不做廉价卖萌
- 正式 assembly 继续按北京区 `OSS + 云剪 cloud-only` 理解

## 展示主线
- 路由画像：api_human_local_footage_light_ppt_cloud_editing
- 主策略：hybrid
- 路由理由：开头和结尾继续用 API 卡通 IP 做判断与收束；中段把豆包的 3 个真实工位拆开给用户看；口诀单独压成一张轻卡，避免卡片反过来抢掉证据。

## Hook
你是不是天天用豆包，却越用越累？不是它没用，是你一直在让它瞎写。

## 结尾落点
你今天先抄一条去试。想要我把这 3 套复制模板整理好，评论区扣豆包，我发你能直接粘贴的版本。
