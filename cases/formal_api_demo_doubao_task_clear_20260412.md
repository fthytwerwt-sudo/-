# 主题
很多人不是不会用 AI，是任务还没收清；先别让它直接写，先把这次到底要交什么收清

## 基础参数
- 总时长：54秒
- 视频比例：9:16

## 目标场景
AI 职场任务现场拆解 / AI 问题点破

## 目标用户
想用 AI 提效、想少走弯路的职场人和轻创业人群；既要能扩散，也要保留转化价值。

## 全局质量要求
- 开头必须由 API 真人承担进入和问题点破，不新增真人实拍出镜
- 中段主体必须由用户录制素材承担，且能看见“旧状态 → 压清动作 → 新状态”的差值
- 中段不能只是聊天录屏占位，静音看也要能读出“先别直接写，先收清任务”
- 结尾固定用单屏 `steps_card` 收束 3 个判断点，不抢走主叙事
- 必须出现一句能直接开口的起手话，用户看完知道自己现在先做什么
- 整条不能像 demo、说明书、课件或硬拼接
- 正式 assembly 继续按北京区 `OSS + 云剪 cloud-only` 理解

## 展示主线
- 路由画像：api_human_local_footage_light_ppt_cloud_editing
- 主策略：hybrid
- 路由理由：API 真人负责开头判断，豆包真实录屏承担“把一句糊话收成可交付任务”的主证据，结尾只用一张 steps_card 把起手句和 3 个判断点压实。

## Hook
很多人不是不会用 AI，是任务还没收清。你越急着让它直接写，越容易拿到一版看起来都对、但根本交不了的东西。

## 结尾落点
你现在先别让 AI 直接写。先开口这句：我现在有点乱，这件事今天得交。你先别直接写，先帮我收清这次到底要交什么。然后只盯 3 个判断点：给谁看，最怕哪里翻车，手里什么能直接用。
