# 主题
把 AI 项目推进卡点压成“API 真人 + 用户本地录制素材 + 少量信息卡 + 云端剪辑”的正式主线

## 基础参数
- 总时长：15秒
- 视频比例：9:16

## 目标场景
AI 项目讲解

## 目标用户
正在推进 AI 项目或 AI 工作流，手里有真实录屏素材，但希望人物判断和收束由 API 真人承担，不再把整条片子做成纯 PPT demo 感的人

## 全局质量要求
- 开头 3 秒必须先用 API 真人判断命中
- 中段必须让用户本地真实过程或真实录屏承担证据感
- 少量 PPT / 图片只负责关键词显影和结果句收束，不抢主体
- 配音不能明显像系统播报
- API 真人必须走真实可调用的人像检测 + liveportrait 路径
- 云端剪辑继续作为正式主路径

## 展示主线
- 路由画像：api_human_local_footage_light_ppt_cloud_editing
- 主策略：hybrid
- 路由理由：API 真人负责开头判断和结尾收束，用户本地录制素材负责过程证据，少量信息卡只做关键词显影和结果句，最终统一进云端剪辑。

## Hook
AI 项目推进不动，很多时候不是缺工具，是你手里还没有一条别人真能接住的链路。

## 结尾落点
先让 API 真人把判断说清，再把用户真实过程录下来，最后只补一张轻量结果卡，再上云端去剪，片子才不会像 demo。
