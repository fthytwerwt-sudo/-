# 结果看板模板

## 1. 这份模板的定位

结果看板不是整个复盘系统的主骨架。

它只负责做结果层证据聚合，不替代：

- 单条视频记录
- 问题层诊断
- ChatGPT 最终判断拍板

一句话：

看板只汇总结果，不单独下最终结论。

## 2. 使用原则

- 一条视频占一行
- 先保证字段可回看，再谈横向比较
- 若缺字段，直接标注缺失，不补脑
- “这个结果更像哪一层的问题”这一列只写初步归类，不写最终结论

## 4. 推荐填写说明

- `主场景` 优先使用：
  - `AI 项目讲解 / AI 方法分享 / AI 学习实操 / AI 案例拆解`
- `主结构` 优先使用：
  - `问题 → 动作 → 结果`
  - `现象 → 原因 → 做法`
  - `提问 → 拆解 → 结论`
  - `旧方式 → 新方式`
  - `结论先行 → 过程补充`
- `这个结果更像哪一层的问题` 推荐只写：
  - `场景层 / 结构层 / 心理机制层 / 画面表现层 / 字幕与配音节奏层 / 待判断`

## 5. 当前一句话规则

看板只做证据聚合，不抢诊断主骨架；真正的复盘仍要回到单条记录和问题层判断。
