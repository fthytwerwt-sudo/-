最近网上很多人在聊 Codex。 一开始大家聊的是“这东西怎么用”。 但聊着聊着，第二个问题就来了： Codex 到底能不能帮普通人赚钱？ 我先说结论。 能。 但不是你打开它，钱就自己来了。 我用了 Codex 差不多 3 个月。 不夸张地说，我应该算是它的忠实用户了。 我现在基本每天都在用它。 而且我越来越确定一件事： Codex、Claude Code 这一类东西，肯定会变成未来很重要的生产力工具。 但生产力工具，不等于赚钱机器。 这句话很重要。 我现在主要拿 Codex 跑 3 个项目。 第一个，是抖音电商的 AI 自动化。 选品、商品测试、数据复盘、视频制作，我现在基本都在让 Codex 参与。基本 90% 的事情完成了自动化 很多事情以前我一个人根本顾不过来。 现在至少能先跑起来。 第二个，就是你们现在看到的这个账号。 这个账号里，Codex 主要帮我做视频流程、素材整理、剪辑测试、复盘这些东西。 说实话，我之前更新断断续续，不是因为没东西讲。 是因为这哥们能干的东西太多了，我一开始反而不知道该从哪讲。 第三个，是我的日常 vlog。 我也在测试，它能不能把那种平时随手记录的东西，慢慢变成一个可以长期跑的内容系统。 听起来是不是挺猛的？ 但我也直接说实话： 这 3 个项目，到现在为止，没有一个真正赚钱。 电商目前也就是保本。 这个账号本来就是分享 AI 使用方式。 vlog 还在测试。 所以如果你问我，Codex 能不能赚钱。 我的真实答案是： 它不能替你赚钱。 但它能让你一个人，同时试更多赚钱的可能。 以前很多事，至少要一个小团队才能做。 有人选品。 有人剪视频。 有人看数据。 有人复盘。 有人整理素材。 有人做下一版测试。 现在我一个人，靠 AI 可以把这些事情先撑起来。 当然，不是撑起来就等于赚钱。 方向错了，还是不赚钱。 产品不行，还是不赚钱。 内容没人看，还是不赚钱。 转化不成立，还是不赚钱。 Codex 解决的是执行成本。 不是赚钱的结果。 但这已经很夸张了。 因为以前最大的问题，不是没有想法。 是想法太多，但一个都跑不动。 现在至少变成了： 我可以先跑一个。 不行再换。 再不行，再换。 单个项目不赚钱，那我就用 AI 把成本压低，多试几个项目，多博几个概率。 这才是我现在真正用 Codex 的方式。 所以我不想把它讲得很玄。 也不想像网上有些内容，要么把它吹成印钞机，要么直接把小白劝退。 我自己之前就是做自媒体的。 我也不是程序员。 我也是一点点摸过来的。 我的感受是： 它确实没那么简单。 但也真的没网上说得那么吓人。 如果你对 Codex、AI 自动化、可以直接在评论区告诉我。 你们想看我拍哪个，我就先讲哪个。 我后面会尽量用真实项目给你们讲。并且诚实的给大家讲清楚 能做到哪一步，我说哪一步。 做不到哪一步，我也直接说。
