# 文案结构改版包模板 copy revision package template

## 文件定位

- 本模板用于把复盘结论转成下一版可执行文案结构。
- 它不是最终脚本。
- 它不是成片执行单。
- 它是 ChatGPT 最终落稿前的结构包。

## 必填字段

1. previous_video_id（上一条视频编号）
2. sample_decision（样本判断）
3. main_problem_layer（主问题层）
4. revision_target（本轮只改一个变量）
5. keep_unchanged（保持不变项）
6. core_judgment_sentence（核心判断句）
7. safe_title_options（安全标题候选）
8. opening_hook（开头钩子）
9. block_structure（block 结构）
10. segment_carrier_map（segment 承载分配）
11. footage_requirement（录屏素材要求）
12. evidence_points（证据点）
13. risk_word_replacements（风险词替换）
14. ending_action（结尾动作）
15. summary_card_copy（总结卡文案）
16. prompt_tail_card_copy（Prompt 引用尾卡）
17. pre_publish_risk_check（发布前风险检查）
18. done_when（完成标准）
19. blocked_if（阻断条件）

## 默认结构

- 核心判断句：
- 标题候选：
- 开头 3 秒：
- 问题点破：
- 录屏证据段：
- 结果差证明：
- 结构整理：
- 结尾收束：
- Prompt 引用尾卡：
- 发布前风险检查结果：

## 禁止

- 不得只写标题，不写结构。
- 不得只写结构，不写录屏承载。
- 不得只写安全词替换，不写核心判断。
- 不得把文案草案写成最终发布稿。
- 不得跳过 ChatGPT / 用户确认直接进入成片执行。
