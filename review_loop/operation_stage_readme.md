# 正式运营阶段说明 operation_stage_readme

## canonical stage

当前《视频工厂》处于：

- `formal_operation_active（正式运营中）`
- `data_driven_operation_iteration（数据驱动运营迭代）`
- `data_driven_operation_loop（数据驱动运营闭环）`

## default route

后续用户提供截图、平台数据、评论、私信或咨询记录时，默认走：

1. `operation_data_intake（运营数据录入）`
2. `operation_review（运营复盘）`
3. `operation_next_variable_decision（运营下一变量判断）`

旧 `gray_test_data_intake` 只作为 2026-05-15 以前的 historical / legacy alias。

## boundary

正式运营只代表项目进入真实发布与数据回流阶段，不代表内容成功、商业验证成立、数据飞轮跑通、多 agent runtime 长期稳定，也不代表当前数据目标锚点 ready。
