# 04_录制计划

## 当前录制减负原则
- 人只录 4 段核心素材
- AI 先把 `场景简报 -> 工作包正文 -> Prompt 包 -> 结果差提示` 整理出来
- 开头人物壳、结尾总结壳、`Prompt 引用尾卡` 的视觉提示词由 AI 先给

## 4 段核心录制素材

### 核心素材 1
- 素材名：原始糊话输入
- 目标：证明原始问题真的模糊
- 必出镜：原始糊话 / 原始输入

### 核心素材 2
- 素材名：主 prompt 执行过程
- 目标：证明主 prompt 如何压出第一版结构
- 必出镜：主 prompt 输入结构 + 第一版输出

### 核心素材 3
- 素材名：修结果 prompt 补强动作
- 目标：证明修结果 prompt 如何把“看起来对”修成“更像能交”
- 必出镜：修结果动作 + 补强后的关键变化

### 核心素材 4
- 素材名：前后差值与总结收束
- 目标：证明前后差值与最终收束
- 必出镜：普通输入 vs 工作包输入差值 + 总结句

## 非录制项
- 开头人物壳：Minecraft-inspired 原创体素方块风
- 结尾总结壳：Minecraft-inspired 原创体素方块风
- `Prompt 引用尾卡`：由 AI 先生成结构和提示词，再由人决定是否微调
