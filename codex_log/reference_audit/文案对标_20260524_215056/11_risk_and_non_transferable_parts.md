# 11 Risk And Non-Transferable Parts｜不可迁移 / 风险清单

## cannot copy

1. 不能照搬原始文案，包括“高级AI军师”“专家模式”等具体第三方产品包装句。
2. 不能照搬人物、头像、账号、平台 UI、搜索页、互动数据、合集标签。
3. 不能照搬 BGM、音效、字幕样式、字体、标题卡皮肤、图标、表情包。
4. 不能复用视频里的买房 / 装修 / 论文研究文档内容和图片素材。
5. 不能把竖屏平台录屏样式当成《视频工厂》横屏正式运营默认交付样式。

## platform / copyright / privacy risks

- 平台 UI 和账号信息长期可见；只能做结构分析，不可作为公开素材。
- 右侧点赞/收藏/评论等数据属于第三方视频信息，不可拿来背书新第四期。
- 原视频人物、头像、可能的产品/品牌元素均有版权和肖像风险。
- 音轨不可抽取、不可复用；后续必须使用自制或授权音效/BGM。

## mismatch risks with video factory

- Reference 强调“专家模式”能力，新第四期必须强调“Codex 初筛 + 人工复核”，不能写成 AI 自动选品成功。
- Reference 竖屏且平台 UI 干扰多；《视频工厂》正式运营默认横屏 16:9，必须重排卡片与字幕。
- Reference 的表格/文档密度很高；新第四期如果照搬会导致商品字段看不清。
- Reference 有多个生活案例，新第四期只应围绕“商品卡初筛”一条主线，避免发散。

## claims that need ChatGPT rewrite or downgrade

- “AI 能直接给答案”必须改为“AI 先整理复查项”。
- “专家/军师”必须降级为“初筛助手 / 结构化整理”。
- “结果很强”必须落到候选表字段和下一步复查项。
- “提效”必须说明提效在哪一段：少翻卡、少漏字段、少把风险混进结论。
