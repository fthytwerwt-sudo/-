## subtitle / overlay notes

- 字幕：内容区底部黑底白字，跟随口播；但平台底部 UI 也占空间，证明新第四期应避免字幕压住商品卡字段。
- 判断卡：适合在抽象判断前出现，例如“问题一”“问题二”，让观众知道屏幕证据为什么出现。
- 信息卡：三模式对比非常适合迁移为“手动翻商品卡 / Codex初筛 / 人工复查”。
- 总结卡：建议新第四期结尾用结果卡告诉用户“留下哪些商品 / 为什么 / 下一步核什么”。
- 边界卡：必须加入“不是让 AI 直接选爆品，不是证明能卖，只是把复查项列出来”。
