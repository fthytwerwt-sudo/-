## Reference Copy Voice（一句话）

它的说话方式不是先讲系统，而是先抛一个普通人听得懂的具体问题，再用短判断句、问题卡、表格/文档高亮把复杂能力翻译成可见结果。

## What Makes It Feel Like Video Copy

1. 开头不是定义产品，而是用能立刻理解的称呼、标题和结果感抓住注意力。
2. 它经常用 `不是 A，而是 B` 的隐性对比，把复杂机制压成判断。
3. 细节不是平均铺开，而是在表格、文档、模式对比出现时集中进入。
4. 过渡句很轻，证明句很重；这正好符合 `copy_granularity_mixture_rule`。
5. 判断不只靠口播，它总会找一个卡片或画面来接住判断。
6. 它的爽点不是“AI 很厉害”，而是“混乱问题突然被拆成几步”。
7. 结尾不强迫转化，而是把能力降回普通用户可做的动作。

## Migration Rule For New Fourth Episode

新第四期不能只学结构，还要学说话顺序：先用生活化痛点开口，再在画面出现字段/表格/聊天框时进入颗粒度；判断句要短、稳、有边界，不能一直像报告在解释机制。
