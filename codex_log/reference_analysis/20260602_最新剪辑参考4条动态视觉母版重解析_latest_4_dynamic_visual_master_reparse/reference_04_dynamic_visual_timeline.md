  - `motion_behavior`: 窗口内容硬切，黄线位置变化，PIP偶尔变化。
  - `transition_behavior`: 主持人 reset 后进入长文本证据，证据持续比其他参考更久。
  - `pacing_feel`: 慢一点，像陪读长文。
  - `attention_path`: 黄色高亮 -> 当前文本段 -> PIP -> 字幕。
