# 画布 / 缩放 / 裁切审计

## 观察

- 成片规格为 1920x1080 / 16:9。
- 源素材来自屏幕录制，可能存在不同窗口比例、浏览器边界、FocuSee 或 macOS 窗口背景。
- 成片里左侧 / 顶部灰边不是单一源素材瞬间，而是与隐私边缘遮挡 / canvas safe band 叠加出现。

## 判断

canvas / scale / crop 不是白屏主因，但参与了灰边观感：

- 源素材可能需要 fit/pad 到 1920x1080。
- padding 或 edge mask 背景色偏灰，叠加 redaction 后形成半透明灰层。
- 边缘遮挡策略把 top / left / right / bottom 作为安全区大块处理，导致画面像被套了灰框。

## 下一轮修复方向

- 保持 1920x1080，但不要用灰色 padding 作为可见背景。
- 对源录屏做统一 crop/scale 后，再做局部隐私遮挡。
- 如果必须遮挡边缘账号 / 路径，用不抢眼的小范围 blur 或低存在感遮罩；不要整条灰边。
