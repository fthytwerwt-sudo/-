# card_placement_decision

## card_budget_gate
- 本稿为长稿，预计成片大概率 > 180s。
- 主卡预算：最多 6 张主卡。
- 本轮建议：6 张主卡 + 1 张弱 CTA 行动卡，行动卡可与结尾画面合并，避免卡片过密。

## 主卡建议
1. 开头痛点卡｜LG001
   - card_text: 测错商品，比拍视频更贵
   - visual: V001 商品卡快闪 + V003/V004 结果表短闪
   - reason: 先抓痛点，不先解释系统。

2. 商品字段卡｜LG008
   - card_text: 价格 / 佣金 / 销量 / 评分 / 风险 / 可拍性
   - visual: V001 01:36-02:09 + V003 00:51-01:30
   - reason: 把“AI 初筛”落到可见字段。

3. 候选表字段卡｜LG010-LG011
   - card_text: 候选表：依据 / 风险 / 状态 / 下一步
   - visual: V003 00:33-01:30
   - reason: 说明商品卡进入表格后的判断结构。

4. 结果变化卡｜LG013
   - card_text: 商品卡 -> 3 个方向 -> 4 个复查对象
   - visual: V004 00:00-00:36
   - reason: 给观众一个明确的结果变化。

5. 复查表结果卡｜LG015-LG017
   - card_text: 为什么留下 / 风险在哪 / 下一步核什么
   - visual: V004 00:39-00:51
   - reason: 这是本片最强证据，必须局部放大或补录。

6. 边界卡｜LG019
   - card_text: 不是爆品答案，只是选品初筛
   - visual: 表格背景虚化
   - reason: 防止平台和商业承诺风险。

## 可选弱 CTA 行动卡
- line_group: LG021
- card_text: 先过表，再拍
- usage: 只作为结尾低压收束，可与复查表画面合并，不单独增加一张重卡。

## 不建议的卡片
- 不建议做“自动赚钱”“爆品推荐”“AI 替你拍板”类卡片。
- 不建议把所有字段一次性塞进一张主卡。
- 不建议使用第三方 reference 的卡片皮肤、字体、BGM、人物或 logo。
