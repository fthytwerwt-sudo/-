## 6. DeepSeek 是否真实生成

本轮 `deepseek_actual_participation = false`。

原因：本轮用户明确禁止读取 `.env` 和 API key。当前 `scripts/deepseek_readonly_explorer.py` 真实调用 DeepSeek API 时会读取本地 `.env`，因此 controller 在 request 禁止 `.env / secret` 时改为安全跳过 DeepSeek API，使用 `fallback_local_only（本地兜底）`。

这不是 DeepSeek passed，也不代表 DeepSeek 稳定真实供料。
