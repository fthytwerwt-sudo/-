## B. Language Texture（语言质感）

- 偏口语、吐槽型、聊天型，不是正式播音稿。
- 句子里有明显的二人称和社交场景代入，例如 ASR 中的“你会发现”“你们女孩儿能不能”。
- 用词不是精修书面词，而是生活口头词：`一大堆`、`那说啥呀`、`那么抽象`、`啥玩意儿`、`真的是`。
- 表达方式带判断和对比，不是单纯讲知识点：男生和女生的表情包被放成“两个极端”。
- 情绪不是大喊大叫，而是顺着观察一路吐槽，像边刷评论边和朋友吐槽。

## C. Sentence Rhythm（句子节奏）

- 节奏偏快，信息密度高，短时间内连续抛出多个并列对象。
- 开头使用连续并列铺排：老师 / 长辈 / 网友，形成“聊天对象不同 -> 收获不同”的节奏。
- 中段出现判断句：“你会发现……真的是两个极端。”
- 后段用连续短口头反应加速：`那说啥呀`、`那么抽象`、`啥玩意儿`、`真的是`。
- 结尾是直接喊话式问题：`你们女孩儿能不能……？`，带轻吐槽和互动感。
- 停顿方式更像自然讲话里的换气和转念，不像逐字规整播报。

## D. Narrative Move（叙述动作）

- 先用泛化观察铺场：和不同人聊天会有不同“收获”。
- 再快速落到具体对象：网友聊天、表情包。
- 接着给出核心判断：当代男生和女生的表情包是两个极端。
- 然后举对比：男生越来越可爱，女生更抽象。
- 最后用喊话 / 反问收尾：希望女生找一些好看可爱的表情包，评论区已经很乱。
- 结构更接近“观察 -> 对比 -> 吐槽 -> 喊话”，不是“问题 -> 方法 -> 总结”的教学稿。

## E. Emotion / Persona（情绪和人设）

- 说话人像轻松吐槽型朋友 / 博主，不像老师或主持人。
- 距离感近，直接对“你们女孩儿”喊话，像对熟人开玩笑式吐槽。
- 情绪核心是“无奈 + 好笑 + 轻微嫌弃”，不是鸡血、励志或卖货。
- 证据：ASR 中出现 `啥玩意儿`、`真的是`、`能不能`、`评论区都乱的啥样子`，这些词都靠口头反应推动，而不是靠正式解释推动。
- 人设不追求完全中立，保留主观判断；但语气应控制在轻吐槽，避免攻击性升级。

## F. Copywriting Reuse Rules（后续写文案复用规则）

可以学：

- 用“你会发现……”引出一个生活观察。
- 用并列对象快速铺开场景，比如“跟 A 聊 / 跟 B 聊 / 跟 C 聊”。
- 用一个鲜明但口语化的判断做中段核心，例如“真的就是两个极端”。
- 用短口头反应制造真人感：`那啥呀`、`真的`、`我服了`、`怎么会这样` 这类词可以少量出现。
- 用轻反问或喊话收尾，增加互动感。

- 不要把它改成大陆短视频鸡血风。
- 不要改成“今天教你三招”的课程腔。
- 不要写成整齐 PPT 播音稿。
