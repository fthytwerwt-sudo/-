# 总索引与阅读顺序

## 1. 文件定位
本文件是当前统一中文文件集的总索引。

当前目录的目标非常明确：
- 只保留一套中文命名文件
- 不再保留旧英文文件名
- 不再保留 archive 目录
- 不再把同一职责拆成多份重复文件
- 当前主规则仍以原 27 份基础规则为骨架
- 命中 vNext 外壳 / Docker 工作台 / 三层 prompt 时，必须补读新增规则文件
- 命中文案知识库时，再补读 `30-36` 文案知识库补充层
